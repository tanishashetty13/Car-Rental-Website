

🚗 Car Rental Website
A modern, high-performance, and fully responsive Car Rental web application designed to provide a seamless vehicle browsing and booking experience.

**🌟 Overview**
This project is a multi-page React application that allows users to explore a fleet of vehicles with ease. It features a clean, professional aesthetic built with Tailwind CSS, ensuring a smooth user experience across all device sizes—from mobile to desktop.

**🚀 Tech Stack**
Frontend: React.js

Styling: Tailwind CSS (Utility-first CSS)

Icons: React Icons / FontAwesome

Routing: React Router DOM for seamless navigation

**✨ Features**
Responsive Design: Fully optimized for mobile, tablet, and desktop views.
Interactive UI: Smooth transitions and hover effects for a premium feel.

Dynamic Catalog: Browse through different car categories and models.

Modern Navigation: Sticky navbar and intuitive routing between pages.
